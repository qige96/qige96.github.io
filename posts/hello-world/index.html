为什么搭建这个网站

一直想有一个属于自己的空间，用来记录技术学习和思考。之前尝试过各种平台，但总觉得不够自由——要么排版受限，要么担心内容被删，要么广告太多。

用 GitHub Pages + 纯 HTML/CSS 搭建静态网站，有几个好处：

  • 完全免费 — 不需要服务器，不需要域名
  • 内容可控 — 所有内容都是 HTML 文件，存在 Git 仓库里
  • 发布简单 — git push 就自动部署
  • 速度够快 — 静态页面，没有数据库查询

技术选型

  • 纯 HTML/CSS: 不依赖任何框架，完全可控
  • GitHub Pages: 托管，与 Git 工作流无缝集成

后续计划

  • 添加更多项目展示
  • 写一些技术文章
  • 考虑绑定自定义域名
  • 添加评论系统

先这样吧，Hello World！
