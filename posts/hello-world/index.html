为什么搭建这个网站

一直想有一个属于自己的空间，用来记录技术学习和思考。之前尝试过各种平台，但总觉得不够自由——要么排版受限，要么担心内容被删，要么广告太多。

用 GitHub Pages + Hugo 搭建静态网站，有几个好处：

  1. 完全免费 — 不需要服务器，不需要域名（当然也可以买）
  2. 内容可控 — 所有内容都是 Markdown 文件，存在 Git 仓库里
  3. 发布简单 — git push 就自动部署
  4. 速度够快 — 静态页面，没有数据库查询

技术选型

  • Hugo: 静态站点生成器，Go 语言写的，构建极快
  • Ananke 主题: 简洁、响应式、维护活跃
  • GitHub Pages: 托管，与 Git 工作流无缝集成

后续计划

  • 添加更多项目展示
  • 写一些技术文章
  • 考虑绑定自定义域名
  • 添加评论系统（Utterances 或 Giscus）

先这样吧，Hello World！
